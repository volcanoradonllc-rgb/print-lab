## ⚙️ Materials & Print Recommendations

These parts are designed with durability in mind.

**Recommended materials:**
- **ASA** → Best for durability and UV resistant
- **PETG** → Great all-around option  
- **PLA** → Only for temporary or indoor use  

**General settings:**
- Walls: 3–4  
- Infill: 25–40%  
- Layer height: 0.2mm  
- Supports: Only if noted  

> Tip: More walls = stronger parts (often better than increasing infill), decrease print times with .6mm hotend nozzle.
